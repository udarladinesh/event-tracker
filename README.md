# Personal Expense Tracker

This project is a simple web application that helps users track their daily expenses.

The application allows users to add expenses, categorize them, and monitor their spending compared to a monthly budget.

## Features

- Add expenses with category and amount
- Edit or delete existing expenses
- Set a monthly budget
- Automatically calculate total spending
- Show remaining budget
- Display category-wise expense summary
- Data is stored in the browser using localStorage

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run the Project

1. Download the project files
2. Open the file `index.html` in any web browser
3. Start adding and managing expenses
